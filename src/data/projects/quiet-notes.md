A minimal note-taking tool built for rough thinking, not polished output.

## Why I built it

Most note apps try to be too much. I wanted something closer to a plain text file — fast to open, zero friction to write in, no formatting pressure.

## What it does

- Stores short notes and build logs locally
- Markdown support for the times I want basic formatting
- No sync, no cloud, no accounts

## The philosophy

Some thoughts are not meant to become full posts or project docs. They are just staging ground. Quiet Notes is where that staging lives.
